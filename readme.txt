URLを貼るだけでnoteやZennのようなサムネイルつきのリンク、ブログカードを表示できます。

== Description ==

WordPress標準のURL埋め込み機能は、リンク先がWordPressサイトでなければ埋め込むことができません。

そこで、あらゆるリンク先をサムネイルつきのカードでスタイリッシュに表示するためにこのプラグインを作りました。

URLをブロックエディターに貼り付けるだけ。あっという間にブログカードを表示できます。
